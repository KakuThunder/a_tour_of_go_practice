# a_tour_of_go_practice

このリポジトリは **a tour of go** の作業履歴を残したものです、
個人での振り返りのために残すので参考にはならないかもしれません。

## ブロック内訳

* first_block
    Packages, variables, functionsなどの基本的な宣言

* second_block
    for, if, else, switch, deferなどの制御構文
